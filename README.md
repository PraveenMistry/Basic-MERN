### Basic-MERN

##### Frontend (Clint)

In the project directory, you can run:

```JS
npm start
```

[Open](http://localhost:3000) to view it in the browser.


##### Backend (Server)

In the project directory, you can run:

```JS
cd backend
nodemon server
```

Runs the app in the development mode.<br>
[Running on](http://localhost:5000) to view it in the postman.
